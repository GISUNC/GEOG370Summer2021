Homework 6: Make a choropleth or proportional symbol map

Instructions: Make a choropleth or proportional symbol map from vector data and a csv that you cleaned and joined.

Make a map about a topic of your interest. The map must not be of a Census topic, use Census data or use a vector file that already has a joined table. The map can be a choropleth, a proportional symbol map, or both. Upload your map to GitHub and create links to your vector file source, the source to the CSV (agency/website) and to the clean CSV that you joined. The map can be from any place in the world and at any scale, as long as is a choropleth or a proportional symbol map that you originally created by joining data. I will accept proportional symbol maps based on CSV with geometry, as long as the CSV has been cleaned and it shows that you have put some effort into selecting an interesting symbol. Proportional symbol maps need to include other vectors (polygons or lines) with labels to put the map into context. The maps need to include a legend and a north and scale if necessary. In your page include a description of your map and your reasons to select that topic.

You will upload the link to your page on Sakai
